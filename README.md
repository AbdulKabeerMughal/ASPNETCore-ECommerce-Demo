# ASP.NET Core MVC E-Commerce Demo

This is a public demo version of an ASP.NET Core MVC e-commerce project created for portfolio and recruitment purposes.

## Important Notice

This repository does not contain the full production source code.

Around 95% of the original project code has been intentionally removed to protect business logic, private implementation details, payment features, admin modules, database logic, and production-level functionality.

This repository is only shared to show the basic project structure and selected sample files.

## What Is Included

- Basic ASP.NET Core MVC project structure
- Program.cs configuration sample
- Selected Views
- Selected CSS files
- Basic layout structure
- Some demo pages
- Public assets and screenshots

## What Is Not Included

- Complete backend logic
- Full admin panel
- Payment gateway integration
- Order management
- Cart and checkout logic
- Customer account logic
- Database connection details
- API keys
- Production services
- Sensitive business logic
- Complete controllers and models

## Purpose

The purpose of this repository is to help recruiters and technical reviewers understand the project structure, UI layout, and ASP.NET Core MVC setup without exposing the complete commercial codebase.
